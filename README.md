№1 Массивы (Работа с парком машин)

1. Создайте массив, представляющий годы выпуска 50 случайных машин (от 2000 до 2025).
2. Выведите только машины, выпущенные после 2015 года.
3. Посчитайте средний возраст авто.

№2 Коллекции (Управление моделями)

1. Создайте список с названиями моделей машин (например: Toyota Camry, BMW X5). ! Могут быть дубликаты
2. Удалите дубликаты, затем отсортируйте модели в обратном алфавитном порядке, выведите на экран, затем сохраните в Set.
3. Реализуйте проверку: если модель содержит слово "Tesla", замените её на "ELECTRO_CAR".

№3 Equals/hashCode (Сравнение автомобилей)

1. Создайте класс Car с полями:
· VIN,
· модель,
· производитель,
· год выпуска,
· пробег,
· цена.
2. Переопределите equals и hashCode, чтобы две машины считались одинаковыми только при совпадении VIN.
3. Добавьте в HashSet несколько машин (включая дубликаты по VIN) и убедитесь, что дубликаты не добавляются.
4. Реализуйте Comparable<Car> для сортировки по году выпуска (от новых к старым).
